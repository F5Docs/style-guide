# Latin abbreviations

## Guidance

Avoid Latin abbreviations. Spell out the English equivalent instead.
Latin abbreviations are harder to translate, unfamiliar to global
audiences, and easy to misuse.

## Examples

**Do:**
> For example, you can set up rate limiting, health monitors, and
> persistence profiles.

> In other words, the pool member is unavailable.

> Set up a virtual server, pool, and so on.

**Don't:**
> E.g., you can set up rate limiting, health monitors, and persistence
> profiles.

> I.e., the pool member is unavailable.

> Set up a virtual server, pool, etc.

## Notes

Use these replacements consistently:

| Avoid | Use instead |
|-------|-------------|
| e.g. | for example |
| i.e. | in other words |
| etc. | and so on |
| vs.  | versus, or (recast the sentence) |
| via  | through, by way of |

Exception: Security advisory articles and vulnerability release notes
may retain Latin abbreviations where they appear in standards or CVE
references.
