# UI terms

## Guidance

Most of the time, avoid referring to UI elements. Instead, describe
what the user needs to do. When you do refer to UI elements, use the
terms in this guide.

## Notes

Use "select" for all UI interactions — buttons, checkboxes, tabs,
list items, and icons. See [Click vs. Select](click-vs-select.md)
for more detail.

UI element terms:

| Term             | Guidance                                                                    | Example                                                                                      |
|------------------|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| button           | refer to the label directly                                                 | "Select the **Submit** button" → "Select **Submit**"                                         |
| checkbox         | refer to the label directly                                                 | "Select the **Enable** checkbox" → "Select **Enable**"                                       |
| dialog box       | pop-up window (MyF5), popup screen (TechComm)                               |                                                                                              |
| drawer           | panel                                                                       | "The drawer expands" → "The panel expands"                                                   |
| drop-down        | refer to the label directly, or use as an adjective before "list" or "menu" | "From the **Policy** drop-down" → "Under **Policy**" or "From the **Policy** drop-down list" |
| field            | box                                                                         | "Enter a value in the **IP Address** field" → "Enter a value in the **IP Address** box"      |
| grayed out       | unavailable                                                                 | "The option is grayed out" → "The option is unavailable"                                     |
| GUI, WebGUI      | user interface                                                              | "In the GUI" → "In the user interface"                                                       |
| navigate to      | go to                                                                       | "Navigate to **Local Traffic**" → "Go to **Local Traffic**"                                  |
| radio button     | refer to the label directly                                                 | "Select the **Yes** radio button" → "Select **Yes**"                                         |
| screen           | page                                                                        | "The Overview screen" → "The Overview page"                                                  |
| tab              | On the **[Label]** tab                                                      | "Select the **Advanced** tab" → "On the **Advanced** tab, select **Enable**"                 |
